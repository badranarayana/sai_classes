logical operators:
* Logical operators are used to combine conditional statements:


1) and
   --> Returns True if both statements are true
     x < 5 and  x < 10
2) or
  --> Returns True if one of the statements is true	
     x < 5 or x < 4
3) not
  --> Reverse the result, returns False if the result is true
    not(x < 5 and x < 10)
